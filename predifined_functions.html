JavaScript String and Math Functions

String Functions

JavaScript provides a variety of functions to manipulate strings:

  • length: Returns the length of a string.
  • toUpperCase(): Converts a string to uppercase.
  • substring(): Extracts a section of a string and returns a new string.
  • toLowerCase(): Converts a string to lowercase.
  • indexOf(): Returns the index of the first occurrence of a specified substring.
  • lastIndexOf(): Returns the index of the last occurrence of a specified substring.
  • slice(): Extracts a section of a string and returns a new string.

Math Functions

JavaScript provides a set of mathematical functions:

  • Math.sqrt(): Returns the square root of a number.
  • Math.pow(): Returns the base to the exponent power.
  • Math.min(): Returns the smallest number among a set of numbers.
  • Math.max(): Returns the largest number among a set of numbers.
  • Math.round(): Rounds a number to the nearest integer.
  • Math.ceil(): Rounds a number up to the nearest integer.
  • Math.floor(): Rounds a number down to the nearest integer.
  • Math.random(): Returns a random number between 0 (inclusive) and 1 (exclusive).
